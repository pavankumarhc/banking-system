🏦 Banking System Web App

A simple and interactive Banking System Web Application built using Python and Streamlit.
This app allows users to perform basic banking operations like checking balance, depositing money, and withdrawing money.

---

🚀 Live Demo

🔗 [https://your-app-link.streamlit.app](https://pavan-banking-system.streamlit.app/)

---

📌 Features

- 💰 Check Account Balance
- ➕ Deposit Money
- ➖ Withdraw Money
- 📜 Transaction History (optional feature)
- 🖥️ User-friendly interface using Streamlit

---

🛠️ Technologies Used

- Python 🐍
- Streamlit
- File Handling (for storing data)

---

📂 Project Structure

banking-system/
│── app.py          # Main Streamlit app
│── Bank.py         # Banking logic (functions)
│── history.txt     # Transaction history
│── requirements.txt
│── README.md

---

▶️ How to Run Locally

1. Clone the repository:

git clone https://github.com/your-username/banking-system.git

2. Navigate to the folder:

cd banking-system

3. Install dependencies:

pip install -r requirements.txt

4. Run the app:

streamlit run app.py

---

🌐 Deployment

This project is deployed using Streamlit Cloud.

To deploy your own:

1. Push code to GitHub
2. Go to Streamlit Cloud
3. Select repository
4. Set "app.py" as main file
5. Click Deploy
